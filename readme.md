## Stack Overflow lookup API using Java Spring Boot 2.3 + VueJS 3

### Requirements:
Java SDK 1.8

Maven

NPM - node package manager

### Running project:

``` bash
# Run JVM
mvn spring-boot:run
```

``` bash
# Compile front-end files
cd front/
npm install
npm run serve
```

Server will start on http://localhost:8080/
